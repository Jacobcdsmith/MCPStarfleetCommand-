## в—Ҳ MISSION BRIEFING

> *"The computer is the most democratic tool ever invented. Its purpose is to do whatever its user requests."*  
> вҖ” *Starfleet Command Directive*

**MCPStarfleetCommand** is an enhanced [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server with a fully interactive **LCARS-styled web dashboard** вҖ” the same aesthetic made famous by the starship *Enterprise*. It bridges Claude Code AI integration (via MCP stdio) with a beautiful browser-based command console, giving you developer tools, file-system ops, system monitoring, and Docker-scale observability from a single unified interface.

## в—Ҳ WARP SPEED DEPLOYMENT

### рҹҡҖ Option A вҖ” Local Launch (Node.js)

```bash
# 1. Install crew
npm install

# 2. Engage engines
npm start                # Production warp
npm run dev              # Debug mode (verbose logs)

# 3. Open LCARS console
open http://localhost:3001
```

### рҹҗі Option B вҖ” Docker Fleet (All Services)

```bash
# Launch the full starship
docker-compose up -d

# Services come online:
#   http://localhost:3001  вҶҗ MCP Server + LCARS Dashboard
#   http://localhost:3000  вҶҗ Grafana Analytics  (change default creds in .env!)
#   http://localhost:3002  вҶҗ Real-time Analytics Engine
#   http://localhost:9090  вҶҗ Prometheus Metrics
#   http://localhost:8080  вҶҗ cAdvisor Container Monitor

# Hail all stations
docker-compose logs -f

# Stand down
docker-compose down
```
