## рҹ–– Mission Briefing

**MCPStarfleetCommand** is a production-ready **Model Context Protocol (MCP) server** wrapped inside a fully-immersive **LCARS-style Starfleet Command dashboard**. It equips Claude AI with **41 powerful tools** spanning git operations, file management, system diagnostics, security auditing, network forensics, and more вҖ” all accessible through a stunning Star Trek-inspired browser interface or directly via the MCP protocol.

> *"The needs of the many outweigh the needs of the few вҖ” and so does a well-instrumented MCP server."*

## рҹ“Ҡ Observability & Metrics

```promql
mcp_tool_executions_total              # Lifetime tool execution count
mcp_tool_execution_duration_seconds    # Latency histogram (p50/p95/p99)
mcp_active_users                       # Concurrent active sessions
mcp_error_rate                         # Percentage of failed executions
```

### Example Queries

```promql
# Top 10 most-used tools
topk(10, mcp_tool_popularity)

# 95th-percentile latency
histogram_quantile(0.95, rate(mcp_tool_execution_duration_seconds_bucket[5m]))

# Rolling error rate
rate(mcp_tool_executions_total{status="error"}[5m]) * 100
```

## рҹ”’ Security

- вң… **Input validation** on every tool parameter
- вң… **Path traversal prevention** via `path.resolve` guards
- вң… **Command output limits** (10 MB buffer cap)
- вң… **Error message sanitization** вҖ” no stack traces to clients
- вң… **Non-root containers** вҖ” all Docker services run as UID 1001
- вң… **Rate limiting** вҖ” Nginx enforces 100 req/s per source IP
- вң… **Network isolation** вҖ” services communicate only within `mcp-network`

### Pre-Production Checklist

```bash
# 1. Change the Grafana password in .env
GF_SECURITY_ADMIN_PASSWORD=your-strong-password

# 2. Set a Redis password in redis.conf
requirepass your-redis-password

# 3. Enable TLS вҖ” uncomment HTTPS block in nginx/nginx.conf
#    and place certs in nginx/ssl/

# 4. Restrict firewall вҖ” expose only ports 80 and 443 publicly
```
